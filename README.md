Aplicación web que permite buscar y generar recetas de bebidas mediante el consumo de APIs e integración de funcionalidades tipo IA

Problema que resuelve:
Los usuarios que buscan recetas de bebidas suelen depender de múltiples fuentes o contenido poco personalizado. Esta aplicación centraliza la búsqueda y genera recomendaciones de forma rápida e interactiva.

Tecnologías utilizadas:
React
Consumo de APIs REST
integración con servicios de IA
CSS / Tailwind
